# EasyCartBilling - Android

A simple billing application built with Jetpack Compose
and Room Database for managing customer bills and cart items.

## Technologies Used
- Kotlin
- Jetpack Compose
- Room Database

## Features
- Add Customer Name and Mobile Number
- Add Items to Cart
- Generate Bill

## Key Learning
- How to use Room Database in Android
- How to build UI with Jetpack Compose
- Local data storage without internet
